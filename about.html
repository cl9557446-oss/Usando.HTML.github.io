Aprendiendo Estructura y Semántica

La importancia del código limpio

Este sitio web ha sido diseñado aplicando los conceptos de HTML5 Semántico. A diferencia del HTML antiguo, que utilizaba etiquetas genéricas para todo, el HTML moderno utiliza etiquetas que describen el significado del contenido que contienen.

¿Qué etiquetas estamos utilizando?

Para este proyecto, hemos implementado una jerarquía clara:

  • <header>: Contiene el título principal y la navegación global.
  • <nav>: Define el bloque de enlaces de navegación.
  • <main>: Envuelve el contenido principal y único de la página.
  • <section>: Organiza el contenido en temas específicos.
  • <footer>: Proporciona información de autoría y derechos.

Beneficios del Desarrollo Semántico

Escribir código de esta manera no solo facilita el mantenimiento para otros desarrolladores en GitHub, sino que también mejora la accesibilidad para lectores de pantalla y ayuda a los motores de búsqueda (SEO) a entender mejor de qué trata nuestro sitio.

Esta práctica es el primer paso para dominar el desarrollo web profesional, sentando las bases antes de profundizar en estilos avanzados con CSS y lógica con JavaScript.
